# CH-to-BS
Convert a Clone Hero `notes.chart`'s properties into an `Info.dat` file

### Input
```
Chart - TheGoodBoi
├── notes.chart
├── album.png
├── song.ogg
├── background.png
└── song.ini
```
### Output
```
Map Folder
├── Info.dat
├── song.ogg
└── cover.png
```
